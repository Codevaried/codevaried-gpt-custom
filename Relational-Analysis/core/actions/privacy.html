Política de Privacidad

Última actualización: 01/01/2025

Esta política de privacidad describe cómo se manejan los datos en el uso del agente "Análisis Relacional", que utiliza la API Math.js para realizar cálculos matemáticos.

1. Recopilación de Datos

Este agente no recopila, almacena ni comparte información personal de los usuarios. Todas las operaciones se realizan en tiempo real, y los datos proporcionados por el usuario se envían únicamente a la API Math.js para procesar los cálculos.

2. Uso de los Datos

La única información utilizada es la expresión matemática que el usuario proporciona. Esta información se envía de forma segura a la API Math.js y no se almacena después de completar el cálculo.

3. Seguridad

Los datos proporcionados son procesados mediante conexiones seguras (HTTPS) para garantizar la protección de la información durante su transmisión.

4. Servicios de Terceros

Este agente utiliza la API Math.js como un servicio externo para realizar cálculos matemáticos. Para más detalles sobre su política de privacidad, consulta la Política de Privacidad de Math.js.

5. Cambios en esta Política

Nos reservamos el derecho de actualizar esta política de privacidad en cualquier momento. Los cambios se reflejarán en esta misma página, junto con la fecha de la última actualización.

6. Contacto

Si tienes preguntas o inquietudes sobre esta política de privacidad, puedes contactarnos en: codevaried@gmail.com.
